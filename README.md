# 📚 Платформаи Дарсӣ бо Flask

Ин лоиҳа як веб-барномаи оддӣ мебошад, ки бо истифода аз **Flask** сохта шудааст.  
Барнома рӯйхати дарсҳоро нишон медиҳад ва ҳар як дарс саҳифаи алоҳидаи худро дорад.

## 🚀 Имкониятҳо

- Намоиши ҳамаи дарсҳо дар саҳифаи асосӣ
- Гузариш ба саҳифаи ҳар як дарс
- Истифодаи шаблонҳои HTML (`Jinja2`)
- Пайваст кардани CSS барои дизайн
- Сохтори сода ва фаҳмо барои омӯзиш
